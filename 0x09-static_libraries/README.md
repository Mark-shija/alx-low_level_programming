# Static Library
* we say Library in programing is a collection of object files that are made available for use in other programs.
They are pre-compiled files containing  functions to avoid repetions.
* Libraries enable us to have separate function file from our main program this enable us to reuse file in different programs as well as ensure compilation speed and file handling practice.

# Types of Library

* we have two types of Libraries
* static Libraries
* Dynamic Libraries
