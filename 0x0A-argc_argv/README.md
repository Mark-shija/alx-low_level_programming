### 0x0A. C - argc, argv
* in this project learned about command line arguments passed int function main.
allowing you to provide inputs to your program when you run it from the command line. 
* argc represents the count of command line arguments, and argv is an array of strings (char pointers) containing the actual command line arguments. argv[0] always holds the name of the program itself, and subsequent elements (argv[1], argv[2], and so on) hold the arguments passed from the command line.
